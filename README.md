# Validador de Palpites de jogos de futebol

Esse projeto foi idealizado por mim mesmo, a um tempo atrás eu decidi criar um bolão no qual eu coletava os palpites dos meus amigos e comparava manualmente os jogos que palpitamos usando utilizando uma planilha

Inicialmente informamos quantos jogos foram palpitados

Após isso devemos informar quantos gols fizeram os times da casa e de fora de cada jogo(um jogo por vez)

Por fim, informamos na mesma ordem os nossos palpites, para assim, o sistema verificar automaticamente quantos pontos foram feitos
